JavaScript Básico

Aqui damos início a nossa primeira linguagem de programação de fato. Através do Javascript iremos começar a dar funcionalidades e ações para as nossas aplicações. Através dele nossas páginas deixam de ser estáticas.

Em sua história, o javascript surgiu com o intuito de rodarmos programação no navegador mas hoje em dia usamos o JS tanto no Front-End (ReactJS) quanto no backend (NodeJS) quanto no mobile (ReacNative) quanto em desktops (ElectronJS). Ou seja, usamos JS para tudo. Mas para darmos passos grandes e entrarmos em frameworks, precisamos ter consolidado bastante os seus fundamentos da linguagem. De nada adianta aprender o framework sem antes saber dominar a linguagem em si.

Sintaxe

É a forma na qual escrevemos texto de uma maneira correta. Toda linguagem possui sua sintaxe. Boa parte dos erros dos iniciantes são causados pela sintaxe. Um simples ponto dentro do universo JS fora do seu devido lugar, pode fazer com que a aplicação inteira pare de funcionar.

Maneiras de executar o JavaScript

Existem basicamente três formas que veremos aqui de executar o JS.

  • Navegador - Podemos executar dentro do próprio browser apertando a tecla F12. Assim abriremos o DevTools onde encontraremos o Console, onde poderemos rodar nossos códigos.
  • CodePenIO - Site externo que nos proporciona um ambiente HTML CSS E JS. Ótimo para estudos de forma clara e limpa.
  • VSCode - Podemos abrir uma tag script dentro de qualquer arquivo HTML. E entao basta abrirmos ele no navegador e ver ele rodando.
    • Como boas práticas, geralmente é indicado criar um arquivo SCRIPT externo, e em seguida linkamos da mesma forma com qual fazemos com o CSS, utilizando a tag script.

Comentários

Para realizarmos comentários em um ambiente JS temos duas formas possíveis:

  • // - Aqui é utilizado quando queremos comentar apenas uma linha.
  • /* - Aqui abrimos a "tag" , e tudo ficará comentado até o final ou até seu fechamento */.

Tipos de dados no JS

O iníncio de tudo no JavaScript ou em qualquer outra linguagem de programação é aprendermos a escrever corretamente sua gramática e seu vocabulário. Conforme vamos aprendendo mais palavras, melhor vai sendo o nosso vocabulário, e quanto maior o vocabulário, maior será nossa fluência.

Começaremos com os tipos de dados.

  • String - Cadeia de caracteres.
  • Number - Identificadores de números
    • Inteiros
    • reais
    • NaN
    • Infinity
  • Boolean - Tipo de dado binário, booleano. É um tipo de dado que só tem dois valores. Ou é verdadeiro ou é falso. (True or False).
  • Undefined vs Null - Null é um objeto que existe porém não possui nada dentro dele. Já indefinido é um objeto que não existe.
  • Object - Tipo de dado estrutural. Ele nos permite setarmos pripriedades (atributos) e funcionalidades (métodos). {}
  • Array - Também é um tipo de dado estrutural muito utilizado na programação. Arrays, Vetores, são um grupamento de dados, as famosas Listas. []
