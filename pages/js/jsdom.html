JavaScript - DOM

O que é Dom?

Document Object Model. A Dom é o HTML convertido para JavaScript. É uma API que interage com o HTML. Estrutura de dados do tipo árvore, criada pelo browser, na qual possui suas propriedades e seus métodos. Cada elemento HTML em sua aplicação é um elemento de DOM e através deles que começamos a manipular o DOM da página(pegamos algum elemento da página e fazemos as devidas alterações.)

O JavaScript usa a DOM para se conectar e manipular o HTML. A programação WEB só existe por que existe a DOM.

Por hierarquia temos docuent -> html -> head/body -> e por ai vai diminuindo.

Selecionando Elementos

  • document.getElementByID('id') - Seleciona o elemento pelo ID.
    • Retorna um Elemento.
  • document.getElementsByClassName('class') - Seleciona os elementos pelas Classes.
    • Retorna um HTMLCollection
  • document.getElementsByTagName('tag') - Seleciona os elementos pelas Tags.
    • Retorna um HTMLCollection
  • document.querySelector('SeletorCSS') - pega o primeiro html pelo Seletor , com a escrita igual a do CSS.
    • Retorna um Elemento
  • document.querySelectorAll('SeletorCSS') - ao contrário do querySelector, trás todos os encontrados e não só o primeiro
    • Retorna um NodeList (Consegue aplicar um ForEach)

Manipulando Elementos

Após pegarmos o elemento e jogarmos ele para uma variável element, por exemplo, podemos começar a manipula-los.

  • let novoElemento = document.createElement(li,div)- cria um novo elemento.
  • element.children - mostra os filhos que possuem dentro do elemento.
  • element.innerHtml = "h2,olá,h2," - Retorna todo o conteúdo HTML. Nos permitindo consultar e alterar valores.
  • element.innerText = "Olá Amigo" - Troca o texto interno do seu elemento HTML.
  • element.textContent = "Olá" - Altera o conteúdo de texto do elemento.
  • element.value = "valor qualquer" - Permite adicionar ou alterar o valor dentro de um Input.
  • element.append("(alterado)") - Diferente dos inner, ele só adiciona o elemento em específico, e não altera a página inteira. Ele também só adiciona Textos e não HTML.
  • element.appendChild(novoElemento)- adiciona um novo elemento criado com o createelement.
  • element.after("adicionado depois") - Adiciona um texto após um elemento
  • element.before("adicionando antes") - Adiciona um texto antes do elemento
    • Se ao invés da string entre aspas, colocarmos um novoElemento, ele irá adicionar o elemento HTML como um botão ou uma div por exemplo.

Manipulando Atributos

  • element.getAttribute('type') - Pega o atributo caso exista.
  • element.hasattribbute('placeholder') - Retorna boolean se tem ou não o atributo.
  • element.setAttribute('placeholder', 'valor a ser colocado') - Adiciona um atributo caso não existe com o valor setado, e caso já existe, substitui o valor.
    • No dia a dia você pode ver isso em um input de senha, clicando para mostrar ou ocultar senhas.

Manipulando CSS (Estilos & Classes)

  • element.style.backgroundColor = "blue" - Permite adicionar CSS inline, trocando os - por camelCase.Se já existir, troca, se não existir adiciona.
    • Podemos pegar ou setar as propriedades.
  • element.classList. - Nos retorna um dOMTokenList, que é um array que nos permite uma série de comandos para trabalharmos com Css.
    • .add('active') - ativa uma classe no elemento selecionado.
    • .contais('classe') - procura se tem uma classe.
    • .remove('active') - remove uma classe no elemento selecionado.
    • .toogle('toogle') - Ele procura uma classe, se achar ele remove, se não achar adiciona.

Eventos

A Dom ela é direcionada a eventos. O Js é direcionado a eventos através de clicks, scroll, passar encima de algo.Existem basicamente os eventos de teclado, de mouse, e alguns outros. Para colocarmos eventos, começamos inserindo a tag on diretamente no HTML, para em seguida completarmos com uma das infinidades de eventos que temos a disposição.

Add eventos via JS.

Para isso precisamos selecionar o elemento através das formas já aprendidas, e atribuir a uma variavel, element, por exemplo.

  • element.addEventListener("click",() => (sayHello) - Para isos, precisamos informar dois argumentos. O primeiro é o tipo de evento, e o segundo qual a função que ele irá executar.

Eventos de Teclado:

  • input.onkeydown - ativa ao apertar a tecla, antes de soltar.
  • input.onkeyup - ativa ao soltar a tecla quando voce pressiona.
  • input.onkeypress - imediato, assim que apertar.

Por padrão todos os eventos em inputs enviam para a função um argumento chamado pelos programadores de Event, ou simplesmente E. Através dele conseguimos saber quais teclas foram digitadas dentro do input, para a partir dai já podermos começar a trabalhar com os dados, antes mesmo do usuário submeter o formuário.

  • e.code - informa cada tecla que o usuário digitou.
  • e.key - mais simplório, utilizada mais para saber se o usuário apertou alguma tecla em específico.
  • e.shiftKey - retorna um boolean se o shift estava apertado ou não na hora que o usuário digitou uma tecla.
