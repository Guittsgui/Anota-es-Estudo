JavaScript - DOM

O que é Dom?

A Dom é o HTML convertido para JavaScript. É uma API que intearge com o HTML. Estrutura de dados do tipo árvore, criada pelo browser, na qual possui suas propriedades e seus métodos.

O JavaScript usa a DOM para se conectar e manipular o HTML. A programação WEB só existe por que existe a DOM.

Por hierarquia temos docuent -> html -> head/body -> e por ai vai diminuindo.

Selecionando Elementos

  • document.getElementByID('id') - Seleciona o elemento pelo ID.
    • Retorna um Elemento.
  • document.getElementsByClassName('class') - Seleciona os elementos pelas Classes.
    • Retorna um HTMLCollection
  • document.getElementsByTagName('tag') - Seleciona os elementos pelas Tags.
    • Rtorna um HTMLCollection
  • document.querySelector('SeletorCSS') - pega o primeiro html pelo Seletor , com a escrita igual a do CSS.
    • Retorna um Elemento
  • document.querySelectorAll('SeletorCSS') - ao contrário do querySelector, trás todos os encontrados e não só o primeiro
    • Retorna um NodeList (Consegue aplicar um ForEach)

Manipulando Conteúdos.

Após pegarmos o elemento e jogarmos ele para uma variável element, por exemplo, podemos começar a manipula-los

  • element.textContent = "Olá" - Altera o conteúdo de texto do elemento.
  • element.innerText = "Olá Amigo" - Troca o texto interno do seu elemento HTML.
  • element.innerHtml = "h2,olá,h2," - Permite adicionar código HTML dentro do elemento via JS.
  • element.value = "valor qualquer" - Permite adicionar ou alterar o valor dentro de um Input.

Manipulando Estilos & Classes

  • element.style.backgroundColor = "blue" - Permite adicionar CSS inline, trocando - por camelCase.
    • Podemos pegar ou setar as propriedades.
  • element.classList. - Nos permite uma série de comandos para trabalharmos com Css.
    • .add('active') - ativa uma classe no elemento selecionado.
    • .remove('active') - remove uma classe no elemento selecionado.
    • .toogle('toogle') - Ele procura uma classe, se achar ele remove, se não achar adiciona.
