Tanstack Query

O que é ?

É uma biblioteca que serve para ser um intermediário entre a aplicação e sua requisição feita com o Fetch ou Axios. Porém eles dois já fazem tudo que precisamos, então para que utiliza-los? O TanstackQuery NÃO serve para fazer requisições, e sim para realizar o fetching, sincronizar, atualizar dado de states. Um exemplo prático: Imagine que tenhamos uma lista de usuários, ao irmos para uma outra tela, e retornarmos para essa tela, ela irá fazer novamente uma requisição. Com o Tanstack, ele já terá aquelas informações, e desse jeito evitará requisições.

O primeiro passo é de Instalar a biblioteca. Para isso, utilizamos o comando: npm i @tanstack/react-query.
