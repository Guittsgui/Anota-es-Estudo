O que é Git ?

Git é um sistema que nos possibilita realizar o controle de versão de nosso projeto. A cada modificação que ocorra em seu software, o git irá detectar isso, e assim podemos ir "salvando" diversas versões do nosso código. O git nos permite também retornar a versões antigas de nosso código. É como se fosse uma história, uma timeline. Outra vantagem, é que podemos criar novas branches, que seriam novas timlines.

Agora que já sabemos o que é, voce pode clicar aqui para baixar em sua máquina. E antes de começarmos lembre-se sempre do comando git help em seu terminal, lá será possível ver diversos comandos.

Principais comandos:

Comando git iniciais

  • git initi - Inicia um repositório git dentro do seu novo projeto.
  • git status ::: monitora os arquivos que ainda nao estão mapeados no git..
  • git add . ::: adiciona todos os arquivos a zona de monitoramento.
  • git commit -m "xx" ::: realiza o commit com uma mensagem.
  • git log ::: mostra a lista dos commits realizados no projeto

oi

  • git branch ::: mostra a lista de branchs e a atual.
  • git reset xxxx idcommit ::: volta para uma determinada branch, podendo ter três formas diferentes:
    • -soft volta para o commit desejado, com as modificações presentes e não commitadas. Vai voltar para antes do commit.
    • -mixed basicamente faz a mesma coisa porém não estará preparado para commitar. A diferença é que as modificações devem ser add denovo.
    • -hard volta de forma definitiva para o commit desejado. Tudo feito na frente será perdido de vez.
  • git branch nomeNovoBranch ::: cria uma nova branch com o nome desejado.
  • git checkout nomeDaBranch ::: sai da branch atual e vai para a branch desejada.
  • git push -u origin main - envia para o github.
