Estrutura de Dados

O que é ?

Parte de extrema importância para entender os fundamentos de estruturas de dados. Quanto maior for o nosso conhecimento em estrutura de dados, provavelmente, isso irá refletir diretamente na qualidade de nossos códigos. Na vida profissional iremos lidar diariamente com organização de dados, boas práticas, escrever melhor algoritmos e muitas vezes esse conhecimento pode até ser crucial na hora de uma entrevista de emprego

Estruturas de dados tem haver com a gestão dos dados na aplicação. Pode ser dividdo em tres partes:

  • Modelagem da estrutura
  • Dar vida á estrutura
  • Criar funcionalidades dessa estrutura

Arrays

Array, vetor ou arranjo, é uma estrutura amplamente utilizada e implementada em quase todas as limnguagens de programação. UUma das estruturas mais básicas e simples de ser utilizada.

Acessamos seus elementos pelo seu index, que sempre parte do Zero. Ele sempre respeita a ordem de inserção dos elementos. Aceita valores duplicados. Arrays no JS são dinâmicos e também podem ter diferentes tipos de dados alocados dentro de um mesmo array. No JS arrays já possuem diversos métodos.

Vetor ou Matriz

Vetor multidimensional. Significa que é um array dentro de outro Array, podendo ter muitos níveis.

Stack

A tradução é pilha assim como uma pilha de livros. O último a entrar na pilha é o primeiro a sair(LIFO, last in, first out).

Queue

A tradução é fila. Como uma fila em uma loja ou restaurante. Diferente da pilha, o primeiro a entrar na fila é o primeiro a sair. (FIFO, first in, first out)
